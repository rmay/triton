Shortcuts

Triton doesn't provide any special support for shortcuts. Use any text editor to change your xbindings and the WM Preferences for WM shortcuts.

xbindkeys

Key Combination Action Command
Mod4 + Return Triton terminal triton-terminal
Mod4 + ~ (grave) Triton dmenu launcher triton-dmenu-run
Mod4 + Space Triton rofi launcher rofi -show run
Shift + Mod4 + Return X terminal xterm
Alt + F10 Region screenshot flameshot gui
Alt + F11 Full screen screenshot flameshot full -p ~/Pictures
Alt + F12 Active window screenshot flameshot screen -p ~/Pictures

Window Maker additional shortcuts

Key Combination Action Description
Mod4 + 1 Switch to Workspace 1
Mod4 + 2 Switch to Workspace 2
Mod4 + 3 Switch to Workspace 3
Mod4 + 4 Switch to Workspace 4
Ctrl + Mod4 + 1 Move focused window to Workspace 1
Ctrl + Mod4 + 2 Move focused window to Workspace 2
Ctrl + Mod4 + 3 Move focused window to Workspace 3
Ctrl + Mod4 + 4 Move focused window to Workspace 4
Mod4 + Left Maximize window left half Maximizes the focused window to the left half of the screen
Mod4 + Right Maximize window right half Maximizes the focused window to the right half of the screen
Mod4 + Up Maximize window full Maximizes the focused window
