# Hardware requirements

Low. If you bought anything new in the past 10 years, odds are it'll run Triton, even if all the peripherals may not be supported by FreeBSD.

Take, for instance, all those machines running Windows 10 but can't run Windows 11 because MS reasons. Triton will run happily on anything with a couple of cores, 4 gigs of RAM, and at least 20 gigs of storage.

If you can install FreeBSD on it, then that's half the battle.
